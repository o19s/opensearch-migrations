# Solr to OpenSearch Migration Advisor

## Purpose

This project provides an AI-powered migration advisor that helps teams migrate from
Apache Solr to Amazon OpenSearch Service. It translates schemas, converts queries,
estimates sizing, and generates migration reports through a conversational interface.

## Users

- **Search engineers** migrating production Solr deployments to OpenSearch
- **Platform teams** evaluating OpenSearch as a Solr replacement
- **Solution architects** building migration plans for enterprise customers

## Business Objectives

- Reduce migration planning time from weeks to hours
- Surface incompatibilities early, before they become production incidents
- Provide accurate, verifiable translations (schema, query, config)
- Generate audit-ready migration reports with cost and sizing estimates
