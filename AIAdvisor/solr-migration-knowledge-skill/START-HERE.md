# START HERE

This repo is a working knowledge base for Solr to OpenSearch migration work.
It is part source library, part consulting playbook, part installable skill, and part demo lab.

## Pick One Path

If you are a consultant preparing for client work:
- start with [README.md](/opt/work/OSC/agent99/README.md)
- then read [intake-template.md](/opt/work/OSC/agent99/02-playbook/intake-template.md)
- then use [interview-guide.md](/opt/work/OSC/agent99/02-playbook/interview-guide.md)

If you are contributing or cleaning up content:
- start with [CONTRIBUTOR-GUIDE.md](/opt/work/OSC/agent99/05-working/CONTRIBUTOR-GUIDE.md)
- then check [CONTENT-STRUCTURE.md](/opt/work/OSC/agent99/05-working/CONTENT-STRUCTURE.md)

If you want a worked example:
- start with [northstar-enterprise-demo](/opt/work/OSC/agent99/03-specs/northstar-enterprise-demo/README.md) for the more realistic enterprise sample
- use [techproducts-demo](/opt/work/OSC/agent99/03-specs/techproducts-demo/README.md) for the smallest clean example
- use [drupal-solr-opensearch-demo](/opt/work/OSC/agent99/03-specs/drupal-solr-opensearch-demo/README.md) for Drupal-specific intake and demo material

If you want to run a local demo stack:
- start with [README.md](/opt/work/OSC/agent99/README.md)
- then use [docker-compose.yaml](/opt/work/OSC/agent99/docker-compose.yaml)
- then use [demo_search_stack.sh](/opt/work/OSC/agent99/tools/demo_search_stack.sh)

If you want to use the skill directly:
- start with [SKILL.md](/opt/work/OSC/agent99/04-skills/solr-to-opensearch-migration/SKILL.md)
- then follow the referenced files in [references](/opt/work/OSC/agent99/04-skills/solr-to-opensearch-migration/references/)

## What Not To Start With

- Do not start by reading all of `01-sources/`. That is the raw library, not the curated path.
- Do not assume every example in `03-specs/` is equally canonical.
- Do not treat every draft in `04-skills/references/` as equally mature without checking [CONTENT-STRUCTURE.md](/opt/work/OSC/agent99/05-working/CONTENT-STRUCTURE.md).

## Mental Model

- `01-sources/`: raw material
- `02-playbook/`: consultant tools
- `03-specs/`: worked examples and engagement output
- `04-skills/`: distilled expert guidance for agents
- `05-working/`: contributor tracking and cleanup backlog
